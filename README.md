# Page Loader

**Page Loader** is a command-line utility that downloads pages from the Internet and saves them on a computer.

## For install Page Loader:
```bash
git clone https://github.com/myxtaridse/js-async-project-4.git
cd js-async-project-4
make setup
```

## Usage:
```bash
page-loader -o <outputDir> <targetUrl>
```
